# Desafio DIO — Redução de Custos com Serviços AWS

## Descrição do Desafio

Neste desafio, o objetivo foi criar um relatório técnico identificando **3 serviços AWS** capazes de promover redução imediata de custos para uma empresa fictícia (**Abstergo Industries**), aplicando os conceitos de computação em nuvem explorados durante o bootcamp.

## Serviços AWS Implementados

| Etapa | Serviço AWS | Objetivo |
|-------|-------------|----------|
| 1 | **Amazon EC2 + Auto Scaling** | Elasticidade computacional e eliminação de capacidade ociosa |
| 2 | **Amazon S3 Intelligent-Tiering** | Armazenamento inteligente com migração automática entre camadas |
| 3 | **Amazon RDS Reserved Instances** | Redução de custos em banco de dados com alta disponibilidade |
